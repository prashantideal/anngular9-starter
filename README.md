# Angular 9 Starter
Basic project template with pre-configured:
1) Authentication/Authorization
2) HTTP Service
3) Logged-in vs Non-Logged-in User Layouts
4) Toggling Side Menu
5) Routing
6) Material Starter
7) Moment for Datetime

No need to start from scartch with boiler-plate code, just fork and start building your features.
